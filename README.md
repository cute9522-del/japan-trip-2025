# 日本行程小工具（GitHub Pages）

此專案為「手機優先」的靜態旅遊小工具（不需建置流程）。

## 使用方式（GitHub Pages）
1. 將本資料夾整包上傳到 GitHub Repo（建議 repo 名稱：`japan-trip-2025`）
2. GitHub Repo → Settings → Pages
3. Source 選擇 `Deploy from a branch`
4. Branch 選擇 `main` / folder `/(root)`
5. 儲存後等待 Pages 產生網址即可

## 本機預覽
由於瀏覽器的 CORS 限制，建議用簡易靜態伺服器預覽，例如：
- VS Code Live Server
- 或 Python：`python -m http.server 8000`

然後開啟 `http://localhost:8000`
